# Inverted-Search
Inverted Search engine implemented in C using linked lists and hashing to index words from multiple files and enable fast keyword-based searching.
## Features
- Index creation from multiple text files
- Keyword search with file list output
- Display indexed data

## Data Structures Used
- Linked List
- Hashing

## Concepts Covered
- File handling
- Dynamic memory allocation
- Pointer-based data structures

## How to Compile
gcc *.c -o inverted_search

## How to Run
./inverted_search file1.txt file2.txt file3.txt
